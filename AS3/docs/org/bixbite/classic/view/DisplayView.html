Classpublic class DisplayView
InheritanceDisplayView View Actor Object
Implements IDisplayView
Subclasses DisplayViewContainer

DisplayView, part of Composite pattern of the display list. The basic View extention to work with flash native DisplayObject. Solution built upon Decoration pattern, to wrap up native Objects into MVC structure. Lets you control/expose selected aspects of your application or structure.
