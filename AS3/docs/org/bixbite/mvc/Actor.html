The Actor is a abstract base class for all the mebers of the MVC triad.

Purpose of this class is to provide unified way to register the member within Observer pattern, get unique id and attach default signal into it. As well as provide common set of signal/slot methods for communication

signalproperty
public var signal:ISignal

Every Actor has a Signal attached into it by default. It can be overridden from subclass by Custom Signal implementation that extends Signal. However direct access to this reference is via Signal interface. Since only necessary set of interface methods are controlled by system. In receiver of particular signal if you know the type of the Custom Signal you can actually use it class name to avoid unnecessary castings.
